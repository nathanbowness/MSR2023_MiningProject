## Documentation

* <http://team.ops4j.org/wiki/display/PAXJDBC/Documentation>

## Build

You'll need a machine with Java 6 or Java 7 and Apache Maven 3 installed.

Checkout:

    git clone git://github.com/ops4j/org.ops4j.pax.jdbc.git

Run Build:

    mvn clean install
